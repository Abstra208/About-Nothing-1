
# Web-capture-API

Permet de prendre un site web en photo png

## API Reference

#### pour prendre un sreenshot

```http
  /screenshot?url={URl}
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `URL` | `string` | **Required**. L'url avec *https://* |
